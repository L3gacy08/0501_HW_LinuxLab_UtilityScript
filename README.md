# 0501_HW_LinuxLab_UtilityScript

## What the Script Does

The script creates folders, makes sample files, copies files to a backup folder, moves files, removes a temporary file, and creates a report.

## Commands used

- `mkdir` = creates directories
- `touch` = creates files
- `echo` = prints messages and writes into files
- `cp` = copies files
- `mv` = moves or renames files
- `rm` = removes files
- `ls` = lists directory contents
- `pwd` = shows current directory
- `date` = shows current date and time
- `cat` = shows file contents

## Why use these commands?

These commands help organize files, make backups, delete files, and can make a report.

## How to run the script 😊

Make the script executable:

You can make the script executable with the following command:
`chmod +x script.sh`
